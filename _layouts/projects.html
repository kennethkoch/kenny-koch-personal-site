Notable Projects

  • GenSend

    GenSend is an LLM wrapper for generating emails and other messages using AI.

    github.com/kennethkoch/gensend | A demo is available at gensend.email

    This tool takes shorthand text as input and generates professional sounding emails that can be tuned to match the sender's desired tone and formality level to enable fast and effective communication in a variety of contexts.

    GenSend is powered by the OpenAI API. The interface is built in React, and the back end uses Node.js. GenSend is hosted on AWS.

  • FortySix

    FortySix is a browser-based distributed computing platform.

    github.com/Minierre/fortysix

    FortySix allows users to customize and run genetic algorithms and distribute the computational load across multiple browser clients.

    Anyone can contribute CPU resources to an ongoing computation by opening a URL in their browser.
